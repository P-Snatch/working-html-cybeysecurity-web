Data Security

Data security involves protecting sensitive data from unauthorized access, leakage, loss prevention and data misuse. It usees:
- Data Encryption
- User Access Controls
- Data Classification
- Data Loss prevention
- Security Poloicies and Procedures

Data encyption is where data is encoded to be unreadable to those who arent authorized It uses cryptographoc algorithms to convert data,
encyrption techniques such as hashing are also used, which are usually found when storing passwords. Encyption has been used for thousands
of years to store data, the most famous encryption method is the Caesar Cipher. Encryption is the most common method of securing data
and can range from how secure they are.

Data loss prevention (DLP), prevent the possibility of data breaches by monitoring streams of sensitive data. The solutions can detect and block
unauthorized transfers of data.
